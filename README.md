# Subscription APP - Vue Laravel

## Project setup
```
composer install
```
### install node 

```
npm install
```
### Make migrations 
```
php artisan migrate
```
### Make Seeder

```
php artisan db:seed
```
### Compile style
```
npm run eq
```
